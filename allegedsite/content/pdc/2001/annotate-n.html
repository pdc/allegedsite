SVG Annotations

This is a test of an idea I had to use SVG to annotate an existing raster image, as an alternative to, for example, using nonstandard extensions to TIFF and and a bespoke image viewer in, say, ActiveX format.

This is not a complete example; I have not yet added buttons for zooming and moving to the next page etc. I am pretty sure they are perfectly doable in Javascript.

Were I to be faced with the task of publishing on the web a mass of TIFFs containing annotations, then my suggestion would be to have either (a) a server plug-in that expands a TIFF file in to a virtual directory containing raster data as PNG and annotations in SVG, or else (b) just translate the CD into a directory of such files, since the hard disc space to do this will cost you about tuppence.

Actually there are a couple of problems. First, since SVG and PNG do not have ‘rights management’ built in to them, there is no way to prevent the user from printing the images they display. This is a surprisingly common request. Second, SVG has translucent colours but not the transparent yellow used by Microsoft’s (Wang-supplied) Image Viewer. I could manage that if I am allowed to require that the raster images use transparency to make the ‘paper’ transparent; the yellow rectangles would then be stuck behind the raster.

The random image file I used is one of | Jeremy Dennis’s weeky strips.
