Australian Flags, 1901—2002

These graphics files were created by me, Damian Cugley using SVG and referring to the Australian Flags Act 1953 for information about the size, shape and positions of the features of the flag. Union Jack instructions from Julian Wiseman’s web site. I created my own versions because I wanted some flags to illustrate an on-line quiz (a work project) and all the ones I could find on-line were on sites with complicated copyright notices which made me unsure whether I was supposed to copy them or not.
