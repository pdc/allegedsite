Oxopoly sketch

This is based on Jo Charman’s first sketch of an Oxford-University themed property-trading game.

Obviously I have not done some little things like writing in all the names of the colleges and stuff—the basic shape is there though and adding the text will be easy enough when I next have some time to spend on it... I just wanted to get the basic framework together...
